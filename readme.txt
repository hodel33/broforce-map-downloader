 OVERVIEW
 ~~~~~~~~~~~~~
 The Broforce Map Downloader automates downloading and organizing user-created 
 maps from Steam, perfect for offline play.


 INSTALL
 ~~~~~~~~~~~~~
 1. Download the executable and config.ini
 2. Adjust settings in config.ini to your preference
 3. Run the executable to start downloading maps


 MAP ID
 ~~~~~~~~~~~~~
 Maps are named in the format 'XYZ-123456789-Map Title.bfg', where:
 > 123456789 = Steam Workshop ID 
 > X = Gameplay Type (1-Standard, 2-Puzzle, 3-Story, 4-Experimental, 
   5-Challenge, 6-Deathmatch)
 > Y = Difficulty Level (1-Normal, 2-Challenging, 3-Brotal)
 > Z = Star Rating (0-5, 0 = not enough ratings)
